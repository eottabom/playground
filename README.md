# Playground

HTML5 미니 게임 & 재미있는 웹 콘텐츠 모음.

## 프로젝트

| 이름 | 설명 | 경로 |
|------|------|------|
| 포츈쿠키 🥠 | 쿠키를 깨서 오늘의 운세 확인 | `fortune-cookie/` |
| Mad Box 🤬 | 혼자 욕하고 스트레스 푸는 익명 채팅. 다양한 파괴 인터랙션으로 카타르시스 제공 | `mad-box/` |

## 기술 스택

- HTML5, CSS3, TypeScript
- Canvas API (파괴 효과, 공유 이미지 생성)
- localStorage (데이터 영속)
- 외부 라이브러리 없음 (바닐라)

## 로컬 실행

정적 파일 서버로 루트를 서빙하면 됩니다.

```bash
npx serve .
```
